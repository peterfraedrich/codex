Codex CSV Import

In order to successfully import a comma-delimeated text file into Codex, it must be formatted in a specific way. The following format is this:

[ip address],[nickname],[subnet],[vlan],[location],[notes]

A simgle line would look like this:

10.1.10.1,fileserver,10.1.10.0/24,0,server room,abcd/1234

Each line MUST be in this format for the import to work correctly, however all fields except [ip address] may be left blank.

More sample entries:

192.168.1.1,,,,,,
192.168.1.2,DNS Server,192.168.1.0/24,,,,
192.168.1.100,DHCP Server, 192.168.1.0/24,10,Offsite,this server is currently offsite for maintenance
